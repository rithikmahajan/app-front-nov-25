## Code Type Markers

### Multiple Ways to Identify Code Type:

```javascript
// Primary method (set by our code)
code.codeType === 'invite'  // Invite code
code.codeType === 'promo'   // Regular promo

// Backup methods (from backend)
code.type === 'invite'      // Invite code
code.isInviteCode === true  // Invite code
```

### Filter Logic:

```javascript
// In Invite Screen - Keep ONLY invite codes
const isInviteCode = 
  code.codeType === 'invite' || 
  code.type === 'invite' || 
  code.isInviteCode === true;

return isInviteCode; // âœ… Show if true

// In Cart Screen - Remove ALL invite codes
const isInviteCode = 
  code.codeType === 'invite' || 
  code.type === 'invite' || 
  code.isInviteCode === true;

return !isInviteCode; // âœ… Show if false
```
